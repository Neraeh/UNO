What card is it?

UNO is using IRC Colors (also called mIRC Colors) to display cards. If your client does not support it, toggle the compatibility mode with the !nocolor command.

  • [+2] draw 2 cards - next player must draw 2 cards and skip his turn
  • [R] reverse card - reverse turns order
  • [S] skip card - next player must skip his turn
  • [J] wild card (also called joker) - can be played on any card and let you choose its color when played
  • [+4] wild draw four card - can be played on any card, let you choose its color when played, and next player must draw 4 cards and skip his turn

User commands

To see the command list, you can use the !help command.

Admin commands

These commands are hidden in the !help command because only OPs and users in the access list can use them

  • !exit - close the bot
  • !exit [code] - close the bot with code [code]
  • !update (only on non-Windows platforms) - execute the updateUNO script in the working directory to download and build lastest git sources
  • !kick [user] - kick [user] from the current game
  • !ban [user] - add [user] to the banlist
  • !unban [user] - remove [user] from the banlist
  • !al [add/del] [user] - add/delete user to/from the access list
Users in the banlist are ignored Users in the access list can use all the commands listed above
